# AI Video Generation Tool

## Overview
This project is an AI-based automation tool that generates short videos (30–60 seconds) using trending news content. It extracts trending topics, generates a script, and produces videos with text overlays and relevant visuals.

## What This Project Does
- Scrapes trending news articles or social media topics
- Generates short scripts using AI
- Converts scripts into video format with images and text overlays
- Produces ready-to-share short-form video content

## Purpose
To automate content creation for news-based short videos, reducing manual effort in research, scripting, and video production.
